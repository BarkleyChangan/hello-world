1. 版本说明
   Android 5.0 -> 默认使用jdk1.7
   Android 7.0 -> 默认使用jdk1.8

2. SKD是Android应用的编译器(Software Development Kit)
   添加环境变量: 
   ANDROID_HOME: D:\program\install\64\AndroidSDK
   PATH:         
   %ANDROID_HOME%\tools;%ANDROID_HOME%\platform-tools;
   验证:
   cmd -> adb version

3. 配置SVN
   a. 安装TortoiseSVN时选择command line client tools,这样在bin目录下才能找到命令行工具svn.exe
   b. 在Android Studio中配置TortoiseSVN工具
   File -> Setting -> Version Control -> Subversion -> user command line client,单击右侧的浏览按钮选择本地安装的svn.exe完整路径
   c. 在Android Studio中使用SVN检出项目
   VCS -> Checkout from Version Control -> Subversion,单击Repositories右方的+按钮,在弹出的小窗口中输入SVN仓库地址,点击OK按钮,回到原窗口点击Checkout按钮,把项目检出到本地

4. 常用插件
   File -> Settings -> Plugins -> 下方的Browser repositories...
   a. Android Parcelable code generator
   b. Android Code Generator
   c. GsonFormat
   d. Android Postfix Completion
   e. Android Drawable Importer
   f. Genymotion

5. 安装Genymotion模拟器
   a. 安装VirtualBox 5.1.14.exe
   b. 安装 genymotion-2.8.1.exe
      安装完成后,将 player.exe 、genyshell.exe 、genymotion.exe 替换掉 根目录的文件
   c. 启动genymotion.exe
      添加一个设备
      VirtualBox -> 配置Oracle VM VirtualBox安装路径(D:/program/install/64/VirtualBox/)
      ADB -> 配置Android SDK路径
   d. 下载安装Genymotion插件
      Settings -> Plugins -> Browse repositories... -> Genymotion(下载插件)
      Settings -> Other Settings -> Genymotion(配置安装路径)
   e. 单击Android Studio中的Genymotion图标,在Genymotion Device Manager窗口中,单击New...按钮
   f. Genymotion账号:
      用户名: changan25801@163.com
      密码:   19830217
   g. 选择模拟器Google Nexus5-4.4.4-API19
   h. 选中新建的模拟器,单击Start按钮启动模拟器

6. 设置SDK,JDK,NDK路径
   Project Structure -> SDK Location -> Android SDK location
                                        JDK location: 去掉勾选Use embedded JDK
                                        Android NDK location

7. 导入快捷键
   Settings -> Import Settings -> 选择keymap-shkstart.jar
   快捷键大小写忽略:
   Settings -> Editor -> General -> Code Completion -> Case sensitive
   忽略大小写提示
   completion: None

8. 设置项目文件编码
   Settings -> Editor -> File Encodings -> Global Encoding: UTF-8
                                      Project Encoding:UTF-8
                                      Default encoding for properties files:UTF-8(勾选Transparent native-to-ascii conversion)

9. 设置显示行号和方法间的间隔符
   Settings -> Editor -> General -> Appearance -> Show method separators
                                                  Show whitespaces

10. 颜色
   在XML文件中默认不透明(透明度为FF),在代码中默认透明(透明度为00)
