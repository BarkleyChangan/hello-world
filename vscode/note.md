- 下载路径

    https://code.visualstudio.com  

- 安装注意事项
    取消勾选“将Code注册为受支持的文件类型编辑器”,其余全部勾选  
    
- 安装插件
  
    Chinese 中文插件
    
    Live Server 在页面上右键 -> Open with Live Server
    
    ​    设置自动保存: 文件 -> 首选项 -> 设置 -> Files:Auto Save: afterDelay 1000
    
- 设置

    1. 取消迷你地图: 查看 -> 取消勾选"显示迷你地图"
    2. 取消面包屑: 查看 -> 取消勾选"显示面包屑"
    3. 渲染空格: 查看 -> 勾选"渲染空格" 

- 快捷键
    1. ctrl + / 注释
    2. ! + tab html基本结构
