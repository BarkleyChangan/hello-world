1. 文件编码设置:
    文本文件编码设置: Window -> Preferences -> General -> Workspace -> Text file encoding -> Other：
    JSP文件编码设置:  Window -> Preferences -> Web -> JSP Files -> Text file encoding-> Other：
    properties文件编码设置: Window -> Preferences -> General -> Content Types -> Text -> Java Properties File -> UTF-8 -> Update -> OK

2. 关联jar包以及源文件，doc文件
    windows -> preferences -> java -> installed jres 目录选中你的安装jres单击edit在选中所有的jre system libraries 单击source attachment 将location path 指向src.zip的目录位置保存退出即可
