# redis安装
1. 安装gcc
yum install gcc-c++
gcc -v

2. 解压并安装redis
tar -zxvf redis-4.0.11.tar.gz
make distclean
make
make install
安装路径: cd /usr/local/bin

3. 拷贝配置文件
cp redis.conf /myredis/redis.conf
修改: daemonize yes

4. 启动redis
redis-server /myredis/redis.conf
redis-cli -p 6379
ping

ps -ef | grep redis
lsof -i :6379

5. 关闭redis
SHUTDOWN
exit

6. 开放端口和IP地址
修改redis.conf文件: bind 0.0.0.0

iptables -I INPUT -p tcp --dport 6379 -j ACCEPT
service iptables save
service iptables restart
service iptables status(查看开放端口)

# redis命令
select 0 选择数据库
keys *   查看当前数据库所有的key
flushdb  清空当前库
flushall 清空所有库
dbsize   查看存储的数量
exist k1 是否存在
move k3 2 移动k3到2号数据库
expire key 为key设置过期时间(秒):expire k2 10
ttl key  查看还有多少秒过期-1表示永不过期 -2表示已过期
type key 查看key是什么类型
del key  删除
config get dir 获取启动路径
config get requirepass
save     备份rdb
bgsave   Redis会在后台异步进行快照操作
lastsave 获取最一次save的时间

事务:
discard 取消事务
exec    执行事务
multi   标记一个事务开始
unwatch 取消watch命令对所有key的监视
watch key1 监视一个或多个key,如果在事务执行之前这个key被其他命令所改动事务被打断

主从复制:
info replication
slaveof 127.0.0.1 6379
slaveof no one
